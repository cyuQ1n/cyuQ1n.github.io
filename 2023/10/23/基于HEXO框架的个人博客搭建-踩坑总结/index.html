基于HEXO框架的个人博客搭建-踩坑总结

一：环境准备

如果您的电脑中已经安装了Git和Node.js，那么恭喜您！您可以直接前往第二个步骤：Hexo搭建。

安装Git

Git是一个分布式版本控制系统，用于跟踪文件的更改并协助多人协同工作。通过Git，可以将Hexo博客上传至Github、Gitee等开源平台。Git的具体使用可以参考廖雪峰老师的Git教程。

  • Windows：下载并安装 git。
  • Linux (Ubuntu, Debian)：sudo apt-get install git-core
  • Linux (Fedora, Red Hat, CentOS)：sudo yum install git-core
  • Mac：使用 Homebrew, MacPorts 或者下载 安装程序。

安装完成后，可在命令行中输入git -v或git --version查看Git版本信息。

安装 Node.js

Node.js 为大多数平台提供了官方的 安装程序。对于中国大陆地区用户，可以前往 淘宝 Node.js 镜像 下载。

其它的安装方法：

  • Windows：通过 nvs（推荐）或者 nvm安装。
  • Mac：使用 Homebrew 或 MacPorts安装。
  • Linux（DEB/RPM-based）：从 NodeSource 安装。
  • 其它：使用相应的软件包管理器进行安装，可以参考由 Node.js 提供的指导。

安装完成后，可在命令行中输入node -v或npm -v查看Node.js的版本信息。
